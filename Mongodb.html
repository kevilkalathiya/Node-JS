db.stud_mst.updateOne(
    { stud_id: 5 },

        $set: {
        stud_name: 'Updated Name',
        stud_email: 'updated.email@example.com',
        stud_birthdate: new Date('1995-05-05')
        }

Delete document whose stud_name is “XYZ” Query: 
db.stud_mst.deleteOne({ stud_name: 'XYZ' });
select data with find and aggregate both functionalities Query: 
db.stud_mst.find({});
=> This will return all documents in the collection.

db.stud_mst.find({{_id: 10}});
=> The statement returns the document whose _id is 4.



db.stud_mst.aggregate([
  {
    $match: {
      stud_birthdate: {
        $gte: ISODate("1985-01-01"),
        $lte: ISODate("1995-12-31")
      }
    }
  },
  {
    $project: {
      _id: 0, // Exclude the _id field
      stud_id: 1,
      stud_name: 1
      stud_email:1
    }
  }
]);


=>The $match stage filters documents where
  stud_birthdate falls within a specific date range.

=>The $project stage shapes the output, including
  only the stud_id, stud_name and stud_email fields
  while excluding the _id field.
